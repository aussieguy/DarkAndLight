Dark and Light Themes Made Easy

For far too long, I would hand-code JavaScript every time I needed to implement a dark and light theme. This time, I decided to take a step back and write a hosted library that could be called anytime and anywhere I needed it. It's now two lines of code and easy to implement.

Step one: Insert the JavaScript script element

<script src="http://127.0.0.1:8080/testJS/theme-toggle.js"></script>

Step two: Add something to click on

<a onclick="toggleDarkLightTheme()" id="sunOrMoon"><img src="sun.png"></a>
